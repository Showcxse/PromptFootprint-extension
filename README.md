# PromptFootprint Browser Extension
    This browser extension is a sister project of mine to the PromptFootprint web app

    The extension recognizes when you are on an AI/LLM site (like chatgpt.com) and tracks the carbon output of your prompts

    Works with a pretty good degree of accuracy for chatGPT, claude, gemini, deepseek, grok, and llama

    These numbers and calculations are estimates for informational purposes. Not a scientific audit by any means.

    This is done with a combination of simple math, jstiktoken, and grid data from the Electricity Maps API
